My Projects

Two-Stage Object Detection Model

Imagine equipping existing security cameras with AI eyes trained to spot danger. This project aims to build such a system, tackling the urgent need for better monitoring in public spaces to prevent tragedies. Using machine learning, specifically object detection techniques like modified CNNs and two-stage methods, the goal is to reliably and quickly identify firearms in video feeds. By teaching AI to not only see a gun but pinpoint its location, this system could trigger immediate alerts and safety protocols, offering a proactive defense.

Computer Vision Deep Learning Object Detection CNN PyTorch

Image Segmentation for Portrait Mode

Developed an advanced image segmentation algorithm to separate human subjects from backgrounds in selfies, enabling high-quality portrait mode effects. Implemented encoder-decoder architecture with UNet and skip connections for pixel-precise boundary detection. The model achieves real-time performance on mobile devices while maintaining high accuracy through optimized transposed convolutions and efficient upsampling techniques.

Computer Vision UNet Image Segmentation CNN Mask R-CNN

Translation Model

Imagine Alexa learning French! This project tackles the challenge of teaching Amazon's voice assistant to translate English into French, opening doors to the European market. Forget clunky old translation methods; this dives into the world of Neural Machine Translation, specifically the powerful Transformer architecture. By ditching sequential processing and embracing clever "attention" mechanisms, Transformers understand context and word order better, especially for longer sentences, paving the way for smoother, more natural translations than ever before.

Generative Adversarial Networks (GANs)

Imagine a digital artist (the Generator) learning to draw anime characters from scratch, pitted against a sharp-eyed critic (the Discriminator) trained to spot fakes. This project explores Generative Adversarial Networks (GANs), where these two AI models engage in a creative duel. The artist constantly refines its technique, trying to fool the critic with increasingly realistic, novel anime faces, while the critic gets better at telling real from fake. It's an AI cat-and-mouse game aiming to automatically generate brand-new anime characters, pushing beyond the limits of older techniques.

GANs Deep Learning TensorFlow Neural Networks Computer Vision

NER for Pharmaceutical Adverse Events

Developed a named entity recognition system for identifying adverse drug reactions in pharmaceutical reviews. This project leverages multiple NER approaches including classical methods (dictionary-based, rule-based), machine learning with CRFs, and deep learning with Bi-LSTM-CNN and Bi-LSTM-CRF architectures. The system successfully extracted drug names and associated side effects from patient reviews, creating valuable drug-to-adverse-event mappings that can inform pharmaceutical research and patient safety initiatives.

NLP Named Entity Recognition Bi-LSTM-CRF Healthcare AI Stanza

YOLOv5-Based Object Detection for Tesla Autopilot

In the realm of self-driving cars, where every millisecond matters, this project dared to give eyes to the machine. Like a vigilant co-pilot, a cutting-edge object detection system was crafted using the swift "Single Stage Detectors". This system empowers the car to instantly perceive its surroundings – traffic lights, stop signs, vehicles, pedestrians, and even animals. By overcoming the limitations of slower two-stage detectors, this project achieved real-time perception. The result? A safer, smarter, and faster autonomous experience.

BBC News Classification with RNNs

Developed an intelligent news categorization system for BBC using Recurrent Neural Networks (RNNs) to automatically classify headlines into relevant categories. This deep learning solution processes news headlines as sequential data, capturing temporal relationships between words to achieve nearly 78% classification accuracy. The model was trained on 2,225 articles across 5 categories and implements sophisticated text preprocessing, embedding, and sequence modeling techniques.
